# ✅ URL CORRECTA - Configuración MiniMax OpenAI

## URL que DEBES usar en ChatMCP
```
https://api.minimax.io/v1
```

## ¿Cómo verificar en tu cuenta?

### Opción 1: Documentación oficial
Los resultados confirman que para usuarios **internacionales**, MiniMax usa `https://api.minimax.io/v1` como URL compatible con OpenAI.

### Opción 2: Verificar en tu cuenta
1. Ve a **platform.minimax.io** 
2. **Inicia sesión** en tu cuenta
3. Revisa la **documentación de API keys** 
4. Busca la URL específica de tu configuración

## ¿Por qué esta URL?
- **Es la URL estándar para usuarios internacionales**  
- **Es compatible con OpenAI SDK**
- **ChatMCP puede validar** los endpoints `/chat/completions` y `/models`

## Instrucciones finales
1. **Borras** la URL actual en ChatMCP
2. **Escribes** exactamente: `https://api.minimax.io/v1`
3. **Guardas** la configuración
4. **Pruebas** si el error 404 desaparece

Esta URL debería funcionar para tu cuenta de MiniMax.
